# CaseCobra - A Modern Fullstack E-Commerce Shop for Custom Phone Cases

This project is a fully developed e-commerce platform built from the ground up using Next.js 14 (Next.js).

The application includes a hidden admin dashboard for order management, file uploads with drag-and-drop functionality, and direct customer purchase capability.

The sleek, modern interface is powered by shadcn-ui, and the project is entirely written in TypeScript. Additional features include a completely tailored phone case configurator, authentication integrated with Kinde, and elegant thank-you emails after each purchase. The design is inspired by Apple's configuration aesthetics.

## Features

- Fully developed shop using Next.js 14 from the ground up
- Includes a stunning landing page
- Custom illustrations created by a professional artist
- Hidden admin dashboard for order management
- File uploads with drag-and-drop functionality
- Direct customer purchase capability
- Sleek, modern interface powered by shadcn-ui
- Completely tailored phone case configurator
- Authentication integrated with Kinde
- Elegant thank-you email after each purchase
- Inspired by Apple's configuration design
- Entirely written in TypeScript
